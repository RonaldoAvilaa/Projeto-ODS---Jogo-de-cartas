# 🌱 ODS: Jogo de Cartas

Jogo de cartas educativo inspirado em UNO, desenvolvido como projeto acadêmico para apresentar conceitos relacionados aos Objetivos de Desenvolvimento Sustentável (ODS) de forma interativa, leve e competitiva.

## 📌 Sobre o projeto

Este projeto nasceu a partir de um desafio proposto para o Dia do Meio Ambiente, com a ideia de criar uma experiência digital que unisse aprendizado, sustentabilidade e tecnologia.

A proposta foi desenvolver um jogo simples, acessível pelo navegador, em que os jogadores pudessem aprender sobre temas como reciclagem, água limpa, energia sustentável e educação enquanto participam de uma partida contra bots.

O resultado é um jogo de cartas com mecânicas familiares, inspirado em UNO, adaptado para trabalhar mensagens educativas relacionadas aos ODS.

## 🎯 Objetivo

O objetivo principal do projeto é usar a gamificação como ferramenta de conscientização ambiental e social.

Durante a partida, o jogador interage com cartas temáticas, eventos educativos e mensagens relacionadas à Agenda 2030, reforçando a importância de ações sustentáveis no cotidiano.

## 🃏 Como jogar

- O jogador compete contra dois bots.
- Cada participante começa com 7 cartas.
- É possível jogar uma carta da mesma cor, mesmo número ou mesmo tipo da carta no descarte.
- Cartas especiais podem bloquear jogadores, inverter o sentido da rodada ou fazer o próximo jogador comprar cartas.
- Cartas multicoloridas permitem escolher a próxima cor da mesa.
- Ao ficar com apenas uma carta, o jogador deve clicar em **ODS!**.
- Vence quem terminar primeiro todas as cartas da mão.
